👋 欢迎来到前端学习项目！

这是一个专门为初学者设计的项目，帮助你理解 HTML 和 JavaScript 的基础知识。

📚 本项目包含的知识点

  • HTML 基础：标签、属性、文档结构、语义化
  • CSS 基础：选择器、盒模型、布局、颜色
  • JavaScript 基础：变量、函数、事件、DOM 操作
  • 页面导航：多页面之间的跳转

🗂️ 项目文件结构

前端学习项目1/
├── index.html          ← 你正在看的这个页面（首页）
├── css/
│   └── style.css       ← 所有页面共用的样式
├── js/
│   └── main.js         ← 所有页面共用的 JavaScript
├── pages/
│   ├── about.html      ← "关于"页面
│   └── contact.html    ← "联系"页面（有交互功能！）
└── README.md           ← 启动说明文档
