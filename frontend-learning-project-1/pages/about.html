了解前端开发的核心概念和学习路径

🗺️ 前端学习路线图

前端开发有三个核心技术，建议按以下顺序学习：

1️⃣ HTML — 结构

HTML 是网页的骨架。它定义了页面上有什么内容： 标题、段落、图片、链接、列表...

学习重点：

  • 常用标签（标题、段落、链接、图片）
  • 表单元素（输入框、按钮、下拉菜单）
  • 语义化标签（header、nav、main、footer）

2️⃣ CSS — 样式

CSS 是网页的衣服。它控制页面的外观： 颜色、字体、大小、间距、布局...

学习重点：

  • 选择器（元素、类、ID、后代）
  • 盒模型（margin、padding、border）
  • 布局（Flexbox、Grid）

3️⃣ JavaScript — 行为

JavaScript 是网页的大脑。它让页面能响应用户的操作： 点击、输入、动画、数据处理...

学习重点：

  • 变量和数据类型
  • 函数和条件判断
  • DOM 操作（找到元素、修改内容）

💡 核心概念速查

DOM（文档对象模型）
浏览器把 HTML 文档解析成一棵"树"，JavaScript 可以通过这棵树来找到和操作页面上的任何元素。
事件（Event）
用户在页面上的操作（点击、滚动、输入等）。JavaScript 可以"监听"这些事件并做出响应。
响应式设计（Responsive Design）
让网页能自动适应不同大小的屏幕（手机、平板、电脑）。核心工具是媒体查询（Media Query）。
盒模型（Box Model）
每个 HTML 元素都是一个"盒子"，由内到外分别是：内容 → 内边距（padding）→ 边框（border）→ 外边距（margin）。

🛠️ 推荐开发工具

  • 代码编辑器：VS Code（免费，插件丰富）
  • 浏览器：Chrome 或 Firefox（开发者工具强大）
  • 版本控制：Git（管理代码变更）
  • 学习网站：MDN Web Docs（最权威的前端文档）
