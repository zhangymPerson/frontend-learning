📚 TypeScript 核心知识点

  • 类型标注：给变量、参数、返回值加上类型约束
  • 接口 (interface)：定义对象的"形状"（有哪些属性、什么类型）
  • 联合类型：一个变量可以是多种类型之一（string | number）
  • 类型断言：告诉编译器"我比你更确定这个类型"
  • 枚举 (enum)：定义一组命名常量，避免魔法字符串
  • 类型推断：TypeScript 能自动推断大部分类型，不用手动标注
